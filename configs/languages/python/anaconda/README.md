# anaconda

## Install

```bash
brew install --cask anaconda
```

**Configuration**

Add the follow command in your profile `~/.bashrc` o `~/.zshrc`

```bash
export ANACONDA=/opt/homebrew/anaconda3/bin
export PATH="$PATH:$ANACONDA"
```

**Reload or apply changes**

```bash
source ~/.bashrc
```

o

```bash
source ~/.zshrc
```
