# lazygit

## Install

```bash
brew install lazygit
```

**Configuration**

Add the follow command in your profile `~/.bashrc` o `~/.zshrc`

```bash
alias cat='bat'
```

**Theme**

1. Finding your config directory
```bash
lazygit --print-config-dir
```

2. Copy files

Copy `config.yml` to `<lazygit-config-dir>`


**Reload or apply changes**

```bash
source ~/.bashrc
```

o

```bash
source ~/.zshrc
```
